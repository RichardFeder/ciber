# CIBER Package Documentation

Documentation for the CIBER power spectrum analysis package.

## Structure

- `api/` - API documentation (auto-generated from docstrings)
- `tutorials/` - Tutorial notebooks and guides
- `pipeline_guide.md` - Comprehensive guide to the power spectrum pipeline

## Building Documentation

To build the documentation:

```bash
cd docs
sphinx-quickstart  # First time only
make html
```

## Contents

The documentation covers:
- Installation and setup
- Pipeline overview
- Module reference
- Example analyses
- FAQ
